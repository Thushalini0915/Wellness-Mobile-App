# Wellness App - Android 📱

Wellness App is a modern Android application developed using Kotlin to help users track their daily habits, mood, hydration, and overall wellness.

## ✨ Features

### 🏠 Dashboard
- Personalized greeting
- Daily step tracking
- Habit summary
- Water intake progress

### 📊 Weekly Analytics
- Weekly overview charts
- Habit completion statistics

### ✅ Habit Tracker
- Add / Edit / Delete habits
- Daily progress tracking
- Completion indicators

### 😊 Mood Journal
- Emoji-based mood logging
- Optional note support

### 💧 Water Tracker
- Quick add buttons (250ml, 500ml, 750ml)
- Custom water entry
- Hydration reminder toggle

### ⚙️ Settings
- Dark Mode
- Notifications
- Account management

## 🛠️ Built With
- Kotlin
- Android Studio
- MVVM Architecture 
-  SQLite 
- MPAndroidChart
